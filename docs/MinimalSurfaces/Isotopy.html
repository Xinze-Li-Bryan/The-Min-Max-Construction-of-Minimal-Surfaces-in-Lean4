    Isotopies and Diffeomorphisms #

    定义isotopy（同痕）和相关概念，这是min-max理论的核心。 根据论文Definition 1.1和1.2。

    Type u_2

    Isotopy: 从[0,1]×M到M的连续映射，每个时刻是微分同胚

    • toFun : ℝ × M → M

      底层映射

    • continuous : Continuous self.toFun

      连续性

    • identity_at_zero (x : M) : self.toFun (0, x) = x

      在t=0时是恒等映射

    • is_diffeo_at_t (t : ℝ) : t ∈ Set.Icc 0 1 → True

      每个t时刻是微分同胚（简化）
