    Min-Max Theory for Minimal Surfaces #

    这个模块定义了Pitts min-max理论的核心概念。

    Main definitions #

    • GeneralizedFamily - 广义曲面族
    • SaturatedSet - 饱和集
    • MinMaxSequence - min-max序列
    • m0 - 面积泛函

    References #

    • Colding-De Lellis, Section 3-4
